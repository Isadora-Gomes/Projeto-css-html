O que é HTML?

HTML, criado pelo britânico Tim Berners-Lee, significa "Linguagem de Marcação de Hipertexto" e é o código que define a estrutura básica de um website. Essa estrutura é composta por elementos, ou tags HTML, que informam ao navegador o tipo de conteúdo sendo construído, como títulos, parágrafos, imagens e links.

O HTML não é considerado uma linguagem de programação, já que ele não pode criar funcionalidades dinâmicas. Ao invés disso, com o HTML, os usuários podem estruturar seções, parágrafos e links usando certos elementos.

Tags HTML Básicas

Parágrafos:

< p > É a tag que define um parágrafo.

Listas:

< ul > É a tag que define uma lista não ordenada.

< ol > É a tag que define uma lista ordenada, em que pode ser uma lista numérica ou alfabética.

< li > É a tag que define um item de uma lista, e que é usada dentro de uma lista não ordenada < ul >, lista ordenada < ol >, ou em uma lista de menu < menu >.

Tabelas:

< table > É a tag que define uma tabela, que é utilizada juntamente com outros três elementos: < tr > < th > < td >.

< tr > Define uma linha em uma tabela HTML.

< th > Define uma célula de cabeçalho em uma tabela HTML.

< td > Define uma célula de dados padrão em uma tabela HTML.

Imagens:

< img > É usado para incorporar uma imagem em uma página HTML. Essa tag tem dois atributos obrigatórios:

  • src - Especifica o caminho para a imagem;
  • alt - Especifica um texto alternativo que descreve a imagem, se a imagem por algum motivo não puder ser exibida.

Links:

< a > Define um hiperlink, que é usado para vincular de uma página a outra. O atributo mais importante do elemento < a > é o atributo href, que indica o destino do link.
