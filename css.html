O que é CSS?

CSS é a sigla para o termo em inglês Cascading Style Sheets que, traduzido para o português, significa Folha de Estilo em Cascatas. É usado para estilizar elementos escritos em uma linguagem de marcação como HTML. Ele define como os elementos HTML devem ser exibidos no navegador, controlando o layout, as cores, as fontes e o espaçamento. CSS permite separar o conteúdo da apresentação, tornando o código mais organizado e facilitando a manutenção.

O CSS pode ser inserido diretamente no HTML, dentro de tags (style) ou em arquivos externos vinculados ao documento HTML, proporcionando flexibilidade e reutilização dos estilos em várias páginas.

Seletores:

São utilizados para "selecionar" elementos HTML aos quais os estilos serão aplicados. Exemplos de três tipos básicos de seletores:

Tag: Aplica estilos a todas as instâncias de uma tag HTML específica.

Nesse exemplo, todas as tags < p > terão o texto na cor marrom.

Classe: Aplica estilos a elementos que possuem uma classe específica. As classes são definidas no HTML com o atributo class.

Nesse exemplo, todos os elementos com a classe "elementos" terá o tamanho da fonte definido como 20 pixels.

ID: Aplica estilos a um elemento único que possui um ID específico. Os IDs são definidos no HTML com o atributo id.

Nesse exemplo, o elemento com o ID "texto" terão o fundo azul.

Propriedades e Valores:

Propriedade: Refere-se ao atributo visual que está sendo modificado, como color, font-size, margin, padding, background-color, entre outros.

Valor: Especifica a definição para a propriedade, como blue para a propriedade color, 16px para a propriedade font-size, ou 10px para margin.

Classes e IDs:

As classes são usadas para aplicar estilos a um ou mais elementos. Elas são definidas no HTML com o atributo class, e no CSS antecedidas por um ponto (.), e podem ser reutilizadas em múltiplos elementos para aplicar estilos comuns.

Os IDs são usados para aplicar estilos a um único elemento. Eles são definidos no HTML com o atributo id, e no CSS antecedidos por uma cerquilha (#). Cada ID deve ser único dentro de uma página HTML, garantindo que o estilo seja aplicado a um único elemento específico.

Cascata e Especificidade:

A cascata é o princípio pelo qual os estilos são aplicados aos elementos HTML no CSS. Ela define como diferentes regras de estilo são combinadas e resolvidas pelo navegador para determinar a aparência final dos elementos. Três fatores principais influenciam a cascata:

  1. Origem do Estilo: Estilos podem vir do próprio navegador, do usuário, ou do autor
  2. Especificidade: Regras com seletores mais específicos têm prioridade sobre regras com seletores menos específicos.
  3. Ordem de Aparição: Quando duas regras têm a mesma origem e especificidade, a regra que aparece por último no CSS é a que se aplica.

Especificidade determina quais regras de estilo têm prioridade quando há conflitos. É calculada com base nos tipos de seletores usados.
